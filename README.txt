===========================
Minimal Portfolio Template
===========================

Thank you for purchasing the Minimal Portfolio HTML Template!

This template is a responsive, clean, and minimalist one-page website ideal for developers, designers, and freelancers to showcase their personal or professional work.

---------------------------
Folder Structure
---------------------------
/css           - Compiled Tailwind CSS stylesheet
/js            - JavaScript for interactive components (if any)
/images        - All images used in the template
/fonts         - Custom fonts (if used)
/documentation - Detailed instructions (optional)
/index.html    - Main HTML file

---------------------------
How to Use
---------------------------
1. Unzip the downloaded folder.
2. Open 'index.html' in your browser to view the site.
3. Customize the content inside 'index.html', and styles inside the 'css' folder as needed.
4. Upload all files to your hosting server via FTP or File Manager.

---------------------------
Requirements
---------------------------
- A modern browser (Chrome, Firefox, Safari)
- Code editor (Visual Studio Code, Sublime Text, etc.)
- Basic HTML/CSS knowledge (for editing)
